# Demo recording

Where the README's hero GIF comes from.

## Record it

1. Run the frontend (`npm run dev` in `frontend/`) and open the app.
2. Hit **Cinematic** (bottom-right) for a full-screen, chrome-free frame.
3. Walk the story: **Pipeline** (Play walkthrough) → **Signal Replay** (Auto-play tour) → **Backtests** (drag the cost sliders). ~60–90s.
4. Capture with any screen recorder (ScreenToGif, macOS screenshot, OBS).

## Export it

- Save as `docs/demo.gif`, ~860px wide, under ~10 MB so GitHub loads it inline.
- In the root `README.md`, uncomment the `<img src="docs/demo.gif" ...>` line in the hero.

That's it — the README renders it at the top.
